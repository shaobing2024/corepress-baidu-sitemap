百度收录增强版插件，自动生成百度兼容的站点地图，支持 XML/TXT 格式，百度站长平台自动推送，助力百度搜索引擎优化。

== Description ==

百度收录增强版是一个专业的 WordPress SEO 插件，专门为百度搜索引擎优化设计。它能够百度收录增强版插件，自动生成百度兼容的站点地图文件，并支持百度站长平台的自动推送功能。

### 主要功能

* **自动生成站点地图**：支持 XML 和 TXT 两种格式
* **百度兼容格式**：完全符合百度站点地图规范
* **百度站长平台集成**：支持自动推送 URL 到百度
* **智能内容选择**：可选择包含文章、页面、分类、标签
* **自动更新机制**：文章发布/更新时自动重新生成
* **缓存优化**：减少数据库查询，提升性能
* **管理界面**：可视化配置面板，一键生成
* **robots.txt 集成**：自动添加站点地图链接

### 为什么选择这个插件？

1. **专业针对百度优化**：与其他通用站点地图插件不同，本插件专门为百度搜索引擎设计
2. **百度站长平台集成**：支持自动推送 URL，加速百度收录
3. **智能优化**：自动计算优先级和更新频率，提升 SEO 效果
4. **性能优化**：使用缓存机制，不影响网站速度
5. **易于使用**：可视化管理界面，无需技术背景

### 支持的内容类型

* 文章（Post）
* 页面（Page）
* 分类目录（Category）
* 标签（Tag）

### 站点地图格式

* **XML 格式**：标准站点地图格式，支持完整元数据
* **TXT 格式**：纯文本格式，简单轻量

### 使用方法

1. 安装并激活插件
2. 进入 设置 → 百度地图
3. 配置设置并生成站点地图
4. 在百度站长平台提交站点地图地址

== Installation ==

1. 上传 `corepress-baidu-sitemap` 文件夹到 `/wp-content/plugins/` 目录
2. 在 WordPress 插件管理界面激活插件
3. 进入 设置 → 百度地图 进行配置

== Frequently Asked Questions ==

= 站点地图生成失败怎么办？ =

检查以下几点：
1. 确保 `wp-content/uploads` 目录可写
2. 检查 PHP 内存限制是否足够
3. 查看错误日志获取详细信息

= 百度推送失败怎么办？ =

1. 确认 Token 是否正确
2. 检查网站是否已验证所有权
3. 确认 URL 格式正确

= 如何查看站点地图内容？ =

在浏览器中直接访问：
* `http://yourdomain.com/baidu-sitemap.xml`
* `http://yourdomain.com/baidu-sitemap.txt`

= 站点地图多久更新一次？ =

* 手动生成：点击按钮立即更新
* 自动更新：文章发布/更新时自动更新
* 缓存时间：1小时

= 如何提高百度收录率？ =

1. 保持内容原创和更新
2. 合理设置站点地图优先级
3. 使用百度推送接口主动提交
4. 确保网站结构清晰
